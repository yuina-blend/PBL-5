# slackbot-py
Slackbotのサンプルコードです。

ブログ「みやびのどっとぴーわい」で作り方を紹介しています。

## 使い方
※事前にSlackのAPIトークンの取得が必要です。

Slackbotライブラリをインストールします。  
``
pip install slackbot
``

bot.pyを実行してBotを起動します。

`
python bot.py
`

上記を実行するとSlack上でbotが起動します。
